🌟 Project Overview

Welcome to the Amazon Product Review Sentiment Analysis System 🚀

This project is an advanced Natural Language Processing (NLP) and Machine Learning application that analyzes Amazon customer reviews and predicts whether the sentiment is:

✅ Positive 😊
✅ Negative 😡
✅ Neutral 😐

The system processes thousands of customer reviews using NLP techniques and machine learning algorithms to understand customer emotions and opinions.

This project demonstrates how AI can help businesses analyze customer feedback automatically.

💡 Why This Project is Powerful?

In today’s world, companies receive millions of online reviews.

Reading all reviews manually is impossible ❌

This project solves that problem using:

🔥 Artificial Intelligence
🔥 NLP
🔥 Sentiment Classification
🔥 Data Visualization
🔥 Text Analytics

🖥️ Application Workflow
🔹 Step 1 — Load Reviews

Amazon customer reviews are loaded from compressed JSON datasets.

🔹 Step 2 — Clean Text Data

Reviews are cleaned by:

Removing special characters
Lowercasing text
Removing stopwords
Lemmatization
🔹 Step 3 — Convert Text into Numerical Form

Using:

📌 TF-IDF Vectorization

The text reviews are transformed into machine-readable numerical vectors.

🔹 Step 4 — Sentiment Prediction

The machine learning model predicts:

😊 Positive
😐 Neutral
😡 Negative
